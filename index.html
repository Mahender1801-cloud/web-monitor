Scores any URL server-side (Vercel) — no PageSpeed API key. Grades server-visible signals: response speed (TTFB), page weight, render-blocking resources, compression, SEO & accessibility. It can't paint LCP/CLS/INP (that needs a real browser — your own store's are on the Web Vitals tab). Enter a second URL to compare two sites side by side.

AUTO checks run server-side 5×/day via GitHub Actions and on demand. MANUAL checks are visual — toggle them Pass / Attention after you eyeball the page. Server-only probes (SSL, robots, meta) can't run from the browser, so Run auto checks refreshes PageSpeed live and flags the rest to the next scheduled run.
